## Quick Start
Use the following command to deploy the Cats-API, run the tests and launch the App. Start by cloning the repository:

```
git clone https://github.com/annisaard/Cats-API.git
cd Cats-API
```

Run the following commands to install the required NPM packages:

```
npm install
```

In the project directory, you can run:
```
npm run dev
```
